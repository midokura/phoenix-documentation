# phoenix-documentation

A place to gather deliverable documentation in a single destination.

## Documentation Structure

This documentation repository is organized by target personas:

- [service-operator/](service-operator/) - Documentation for service operators
  - **[IAAS_CONSOLE_CONFIGURATION.md](./service-operator/IAAS_CONSOLE_CONFIGURATION.md)** - Configuration reference for the IaaS Console API Helm Chart.
  - **[GOOGLE_SSO_SETUP.md](./service-operator/GOOGLE_SSO_SETUP.md)** - Provides instructions for operators on configuring Google SSO, which is required for user authentication in the IaaS Console.
  - **[OPERATOR_API_GUIDE.md](./service-operator/OPERATOR_API_GUIDE.md)** - Guide for using the Operator APIs to manage users and tenants.
  - **[GOOGLE_SSO_SETUP.md](./service-operator/GOOGLE_SSO_SETUP.md)** - Provides instructions for operators on configuring Google SSO, which is required for user authentication in the IaaS Console.
- [tenant/](tenant/) - Documentation for tenants
- [user/](user/) - Documentation for users
