Version 1.8 of AI Factory is now available.

## Overview

These release notes describe the revised steps, configuration details, and changes for provisioning and managing an AI Factory cluster under the new release.

This version introduces vpn observability.

<!-- truncate -->

### VPN Observability

This release introduces a new monitoring dashboard for multi-tenant WireGuard VPN environments, providing real-time and historical visibility into peer connections, network traffic, service uptime, and tenant-level usage. Operators can track agent health, reconciliation activity, and user synchronization, and quickly troubleshoot issues using detailed metrics and live VPN server logs.

For more details, see the [WireGuard VPN - Multi-tenant Monitoring dashboard documentation](../docs/service-operator/OBSERVABILITY_DASHBOARDS#8-wireguard-vpn---multi-tenant-monitoring).
