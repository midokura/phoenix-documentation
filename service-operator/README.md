# Service Operator Documentation

This directory contains documentation for **service operators** who are responsible for deploying, configuring, and maintaining the Phoenix infrastructure.

## Target Audience

Service operators are technical personnel responsible for:
- Installing and configuring Phoenix services
- Managing system infrastructure
- Monitoring and maintaining system health
- Troubleshooting system-level issues
- Performing upgrades and maintenance

- [Installation Guide](installation.md) - Step-by-step installation instructions

- [Troubleshooting](troubleshooting.md) - Common issues and solutions
- [Upgrade Guide](upgrade-guide.md) - Version upgrade procedures
