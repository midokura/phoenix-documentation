# Tenant Documentation

This directory contains documentation for **tenants** who manage resources and users within their allocated space in the Phoenix system.

## Target Audience

Tenants are administrators or managers responsible for:
- Managing tenant-specific resources
- Configuring tenant settings and policies
- Managing user accounts and permissions within their tenant
- Monitoring resource usage and quotas
- Administering tenant-level configurations
