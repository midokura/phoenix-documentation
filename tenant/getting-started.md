# Getting Started - Tenant

Welcome to Phoenix! This guide will help you get started as a tenant administrator.

## Introduction

As a tenant administrator, you have access to manage resources, users, and configurations within your allocated tenant space.

## Prerequisites

- Tenant account credentials provided by your service operator
- Access to the Phoenix web console or CLI

## First Steps

### 1. Log In

Access the Phoenix web console at the URL provided by your service operator:

```
https://phoenix.example.com
```

Log in with your tenant administrator credentials.

### 2. Review Your Tenant

After logging in, you'll see your tenant dashboard showing:
- Current resource usage
- Active users
- Recent activity
- Available quotas

### 3. Set Up Your Environment

Configure your tenant settings:
- Review and update tenant profile
- Configure default policies
- Set up notification preferences

### 4. Add Users

Add your first users to the tenant:
1. Navigate to **Users** > **Add User**
2. Enter user details
3. Assign appropriate roles
4. Send invitation

## Key Concepts

### Resources
Resources are the computational and storage assets allocated to your tenant.

### Users
Users are individuals who can access resources within your tenant.

### Quotas
Quotas define the limits of resources your tenant can consume.

### Roles
Roles define what actions users can perform within your tenant.
