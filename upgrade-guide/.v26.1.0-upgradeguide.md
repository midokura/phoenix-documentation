### Remove deprecated inventory keys

- **`ceph.pg_autoscale_mode`**: The key no longer exists in the role. If your inventory has `ceph.pg_autoscale_mode: on/off/warn`, remove it. PG count is now auto-calculated and you can override it with `ceph.pg_count` if needed.
- **`hedgehog.network.rename_interfaces`**: The feature is gone. If your inventory has a `rename_interfaces` list under `hedgehog.network`, remove it. Interface naming is now derived from `hedgehog.networks[0].mac.address`, which you likely already have defined.
- **`hedgehog.network.interface_name`**: Also removed from the schema. If you had this set, remove it — the role no longer uses it.


### Migrate `hedgehog` flat params to structured params

`bridge_name`, `vm_mac`, and `mgmt_nic_pci` still work but are deprecated. Migrate to the `networks` list and `hostdev` dict when convenient.

Old (still works):
```yaml
hedgehog:
  bridge_name: "virbr-libvirt"
  vm_mac: "02:7f:3c:a9:4d:e2"
  mgmt_nic_pci: "pci_0000_01_00_1"
```

New (preferred):
```yaml
hedgehog:
  networks:
    - bridge: "virbr-libvirt"
      mac:
        address: "02:7f:3c:a9:4d:e2"
  hostdev:
    name: "pci_0000_01_00_1"
    address:
      type: "pci"
      domain: "0000"
      bus: "05"
      slot: "00"
      function: "0"
```
