# User Documentation

This directory contains documentation for **end users** who interact with the Phoenix system to accomplish their tasks.

## Target Audience

End users are individuals who:
- Use Phoenix services for their daily work
- Create and manage their own resources
- Interact with the Phoenix UI or CLI
- Consume services provided by the platform

## Documentation Contents

- [Getting Started](getting-started.md) - Quick start guide for new users
- [User Guide](user-guide.md) - Comprehensive guide to using Phoenix
- [Tutorials](tutorials/) - Step-by-step tutorials for common tasks
- [CLI Reference](cli-reference.md) - Command-line interface documentation
- [UI Guide](ui-guide.md) - Web interface user guide
- [FAQ](faq.md) - Frequently asked questions
